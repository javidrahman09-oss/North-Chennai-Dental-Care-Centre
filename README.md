# 🦷 North Chennai Dental Care Centre
### *Where Luxury Meets Oral Healthcare*

A premium, high-performance landing page designed for a dental clinic in **Thiruvottiyur, Chennai**. This project focuses on a "Navy & Gold" visual identity, delivering a sophisticated user experience through modern CSS techniques and smooth interactions.

---

## 🎨 Visual Identity & UX
* **Aesthetic:** High-contrast Dark Mode with Gold accents (`--navy` & `--gold`).
* **Modern UI:** Features **Glassmorphism** on the navigation bar and **Floating UI** elements for real-time clinic status.
* **Typography:** A refined mix of *Playfair Display* for headings and *DM Sans* for high-readability body text.

---

## 🚀 Key Features
* **Interactive Hero Section:** Includes a custom dental-themed SVG and dynamic "Pulse" animations for live status indicators.
* **Treatment Marquee:** A high-speed CSS ticker showcasing the full range of dental services.
* **Smart Service Grid:** 9+ specialized treatment cards featuring interactive hover states and iconography.
* **Conversion Focused:** Fixed **Floating Action Buttons (CTAs)** for one-tap calling and appointment booking.
* **Responsive Engine:** Fully fluid layout that adapts seamlessly from 4K monitors to mobile devices.
* **Performance First:** Zero heavy JavaScript dependencies; powered entirely by semantic HTML5 and CSS3.
