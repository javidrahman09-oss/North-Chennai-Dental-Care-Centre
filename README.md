North Chennai Dental Care Centre - Website
A premium, high-performance landing page for a dental clinic located in Thiruvottiyur, Chennai. This project features a modern "Navy & Gold" aesthetic, smooth scroll animations, and a fully responsive layout.

🦷 Project Overview
This website is designed to provide a luxury user experience while maintaining high functional utility. It includes sections for services, patient reviews, a contact form, and clinic operating hours.

Key Features
Modern Hero Section: Features an interactive dental-themed SVG and floating status cards.

Dynamic Marquee: A scrolling "ticker" highlighting various dental treatments.

Service Grid: Interactive cards for 9+ specialized dental treatments with hover effects.

Responsive Design: Optimized for Desktop, Tablet, and Mobile devices using CSS Media Queries.

Performance Focused: Uses system fonts and CSS variables for fast loading and easy branding updates.

Scroll Animations: Subtle "fade-up" animations triggered as the user explores the page.

🛠️ Technologies Used
HTML5: Semantic structure for SEO and accessibility.

CSS3: * Custom Properties (Variables): For consistent color theming (--navy, --gold).

Flexbox & CSS Grid: For complex, responsive layouts.

Keyframe Animations: For the marquee and hero pulses.

Glassmorphism: Applied to the navigation bar using backdrop-filter.

Google Fonts: Integration of Playfair Display (Serif) and DM Sans (Sans-Serif).
